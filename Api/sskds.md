# LSUDsとSSKDs、オーケストレーション層
APIがどの層をターゲットとしているのかという議論

## オーケストレーション層とは

## LSUDs
Large set of unknown developersの略。
つまり、FacebookやTwitterのような、パブリックなAPIのこと


## SSKDs
Small set of known developers の略。
つまり、自社サービスやクライアントをターゲットとしたAPIを指している。
